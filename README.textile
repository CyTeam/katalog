h1. doku-zug.ch Katalog

Katalog ist eine Katalogsoftware für ein Dokumentenarchiv.

* Webbasierter Kundenkatalog
* Webbasierte Katalogadministration

Katalog basiert auf "Ruby On Rails":http://www.rubyonrails.org und ist als "Open Source":http://www.opensource.org veröffentlicht.
